# LS ELECTRIC — Next.js

[LSEA 디자인](https://www.figma.com/design/FJa9pa866Be2aj5HYV717D/LSEA_%EB%94%94%EC%9E%90%EC%9D%B8) 기반 LS ELECTRIC 웹 프론트엔드 (Next.js 16 App Router).

## Getting Started

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)

| 명령 | 설명 |
|------|------|
| `npm run dev` | 개발 서버 |
| `npm run build` | 프로덕션 빌드 |
| `npm run start` | 프로덕션 서버 |
| `npm run lint` | ESLint |
